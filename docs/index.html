GrandMA 2 Websockets API 3.7.0

The GrandMA 2 Websockets API is the API used by the web remote.

Servers

{deskIp}/?ma=1 ws

The GrandMA 2 console software

deskIp: The IP of the desk we want to control.
DEFAULT: 192.168.0.1
